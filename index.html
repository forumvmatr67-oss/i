📐 Определения

  • 1 ipybyte (IpyB) = 2⁹⁰ байт = 1 237 940 039 285 380 274 899 124 224 байт
  • 1 hpybyte (HpyB) = 1000 × ipybyte
  • 1 gpybyte (GpyB) = 1000 × hpybyte = 1 000 000 × ipybyte

🐍 Код для разработчиков

# Python (GIS)
IPY = 2 ** 90
HPY = 1000 * IPY
GPY = 1000 * HPY

def to_gpy(bytes_val): return bytes_val / GPY
def from_gpy(gpy_val): return gpy_val * GPY

// JavaScript (BigInt)
const IPY = 2n ** 90n;
const HPY = 1000n * IPY;
const GPY = 1000n * HPY;
